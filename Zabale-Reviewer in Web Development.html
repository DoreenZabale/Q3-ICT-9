Title element
- The title element is coded after the head element and specifies a title for the document that can be found in the browser's tab or title bar.
*[title][/title]
HTML Headings
- The HTML headings are mostly the reason why the viewers read the website because it piques their interests. A document contains a main
heading and sub-headings that are coded from one to six; the higher the number, the smaller the fonts.
*[h1][/h1], [h2][/h2], [h3][/h3], [h4][/h4], [h5][/h5], [h6][/h6]
DOCTYPE html Declaration
- The first one to be coded in an HTML file is the DOCTYPE Declaration. It defines this document as HTML5 and serves as information to
the browser about the expected document type.
*[!DOCTYPE html]
HTML Line Breaks
- The HTML Line break helps in creating a new line or a line break, which our enter key cannot do because an HTML file and web browser
ignore white spaces.
*[br]
HTML Paragraphs
- This defines a paragraph or sentence in an HTML document. This is where the majority of the website's contents and sentences are indicated.
It adds a new line after the start and end tags are declared. Text alignment codes can also be used to align it in the center, right, and left side.
*[p][/p]
